# pinterest-mcp-project

A personal project used to connect Pinterest API with a local MCP agent for inspiration and reference. For personal use only.
This project is a first-party, locally executed application that accesses only the owner's authorized Pinterest account for personal use. No data is collected from other users, shared publicly, or transmitted to external servers.


# Privacy Policy

This project ("Pinterest MCP Integration") is a personal, local-use application.

## Data Access
- The application only accesses data from the owner's authorized Pinterest account.
- No third-party user data is collected.

## Data Storage
- Data may be temporarily stored locally on the user's own device.
- No data is uploaded, shared, or transmitted to external servers.

## Data Sharing
- No data is shared with any third party.
- No analytics, tracking, or advertising is used.

## Security
- API keys and tokens are stored locally and are not shared or published.
- Users are responsible for keeping their own credentials secure.

## Purpose
- The application is for personal research and inspiration only.
- It is not offered as a public service or commercial product.
